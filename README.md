## Where Can I See Commits?

You can view the commit history for this repository in several ways:

### On GitHub (Web Interface)

1. **Commits Page**: Visit [https://github.com/antn-cooper/goit-markup-hw-02/commits](https://github.com/antn-cooper/goit-markup-hw-02/commits) to see all commits
2. **From Repository Home**: Click on the commit count (e.g., "10 commits") near the top of the repository page
3. **Specific Branch**: Navigate to a branch and view its commits by clicking "Commits" in the branch view

### Using Git Command Line

If you have cloned this repository locally, you can use these Git commands:

```bash
# View recent commit history
git log

# View compact one-line commit history
git log --oneline

# View last 10 commits
git log -10

# View commits with graphical representation
git log --graph --oneline --all

# View detailed information about a specific commit
git show <commit-hash>
```
